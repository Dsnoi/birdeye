Packageorg.un.cava.birdeye.ravis.globals
Classpublic class GlobalParamsVGraph

This class will hold global params in static variables related to VisualGraph properties. Basically every instantiated component will register itself with this class. graphLayout based components will have to be registered from the application (but it is basically only VisualGraph and the layouter).
