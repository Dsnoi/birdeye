This is a directed edge renderer, which draws the edges with slim ballon like curves that indicate a source. Please note that for undirected graphs, the actual direction of the edge might be arbitrary.

draw()method
public override function draw():void

The draw function, i.e. the main function to be used. Draws a curved line from one node of the edge to the other. The colour is determined by the "disting" parameter and a set of edge parameters, which are stored in an edge object.
