draw()method 
public function draw(vedge:IVisualEdge):void

The draw function, i.e. the main function to be used. Draws a straight line from one node of the edge to the other. The edge style can be specified as XML attributes to the Edge XML Tag Draws an edge. Colours and linestyle can be provided through the XML object associated with the (v)edge. flexible.

Parameters
vedge:IVisualEdge — The edge to draw, it needs to provide all the information required, i.e. locations.
