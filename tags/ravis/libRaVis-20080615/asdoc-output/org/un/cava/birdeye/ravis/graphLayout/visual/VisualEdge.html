Classpublic class VisualEdge
InheritanceVisualEdge flash.events.EventDispatcher
ImplementsIVisualEdge, mx.core.IDataRenderer

A visual edge is just a representation of a graph edge in the VisualGraph context. However, right now the only feature a VisualEdge has is a reference to its Graph Edge and a data object to support it's drawing features. There is a small possibility that this class will be removed.

public function VisualEdge(vg:IVisualGraph, edge:IEdge, id:int, data:Object = null, lview:UIComponent = null, lStyle:Object = null)

The constructor initialiazes the edge and must be preset with almost all parameters.

Parameters
vg:IVisualGraph — The associated VisualGraph.
 
edge:IEdge — The associated Graph Edge.
 
id:int — The internal id of this VisualEdge.
 
data:Object (default = null) — The associated data Object.
 
lview:UIComponent (default = null) — The associated label view.
 
lStyle:Object (default = null) — The line style of the edge, must be an object as associative mapping containing Graphics.lineStyle() parameters.
