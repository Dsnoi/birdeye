Classpublic final class EdgeType

Static final class containing constants for specifying which edges are desired in graph-query methods. Note that constants can be OR'd together using bitwise operations in any combination. For example, EdgeType.UNDIR | EdgeType.SELFLOOP specifies all undirected and self-loop edges. Unlike most of the coraldata.core.api package, this is a concrete class as opposed to an interface because constants can not be added to an interface.


Example
The following code returns all the undirected and self-loop edges incident to a randomly retrieved vertex. 
 
  
  var it:IIterator = graph.getEdges( graph.aVertex() , (EdgeType.UNDIR | EdgeType.SELFLOOP) );
