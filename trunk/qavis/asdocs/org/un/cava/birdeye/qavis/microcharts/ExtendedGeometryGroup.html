Classpublic class ExtendedGeometryGroup
InheritanceExtendedGeometryGroup com.degrafa.GeometryGroup

This class extends the functionalities of the GeometryGroup with tooltips properties and methods. It's therefore possible to define tooltips, their functions and prefix to customize them. If no function is defined for the tooltips, than the dataField value is taken for their content. This class allows creating and positioning the graphics associated with the tooltips.
