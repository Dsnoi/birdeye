colorsproperty
colors:Array  [read-only]

The Micro100BarChart component is used to create 100% Bar microcharts. The basic simple syntax to use it and create an 100% Bar microchart with mxml is:

<Micro100BarChart dataProvider="{myArray}" width="20" height="70"/>

The dataProvider property can only accept Array at the moment, but will be soon extended with ArrayCollection and XML. To insure the proper sizing binding, it's better to use the following properties:

- sizeX: to define the width of the chart;

- sizeY: to define the height of the chart;

It's also possible to change the colors by defining the following Array of int:

- colors: array that sets the color for each bar value. The lenght has to be the same as the dataProvider.

If no colors are defined, than the 100 bar will display different colors based on the default color and a default offset color.

This property sets the colors of the bars in the chart. If not set, a function will automatically create colors for each bar. Implementation
