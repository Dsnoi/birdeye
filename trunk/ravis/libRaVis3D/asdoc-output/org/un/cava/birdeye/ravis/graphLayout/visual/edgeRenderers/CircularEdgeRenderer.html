public override function labelCoordinates(vedge:IVisualEdge):Point

This method places the label coordinates at the functional midpoint of the bezier curve using the same anchors as the edge renderer. Returns the coordinates of the label for the given edge. Different edge renderers might want to specify a different place where to put the label.
