Packageorg.un.cava.birdeye.ravis.utils
Classpublic class GraphicsWrapper

The class wraps the Graphics class so that you can add a fuzz factor around edges. This is so you can mouse over them much more easily. It draws an invisible line underneath the lines that are actually being drawn so that mouse events are caught, but it does not appear any different to the user.
