The draw function, in this renderer draws an Arrow but not straight or curved but only with angles of 90 degrees parallel to the x and y axis. The draw function, i.e. the main function to be used. Draws a straight line from one node of the edge to the other. The edge style can be specified as XML attributes to the Edge XML Tag Draws an edge. Colours and linestyle can be provided through the XML object associated with the (v)edge. flexible.
