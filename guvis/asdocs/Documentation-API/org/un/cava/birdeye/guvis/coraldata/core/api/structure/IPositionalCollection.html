Positional collections (for example, sequences, trees, and graphs) are collections in which elements are related to each other through adjacency information.

A positional collection stores elements at its positions and defines adjacency relationships between the positions (for example, before/after in a sequence, parent/child in a tree). Thus, the model of storage is topological -- i.e., a positional collection is a graph, or some restricted version of a graph. This means that the position at which an element is stored is decided by the user and is arbitrary from the point of view of the collection.
