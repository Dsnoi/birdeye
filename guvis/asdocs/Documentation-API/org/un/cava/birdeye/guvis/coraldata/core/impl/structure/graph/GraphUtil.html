unsplitEdge()method 
public static function unsplitEdge(v:IVertex, edgeElement:Object):IEdge

Transforms edge-vertex-edge into a single edge. The vertex must be of degree 2. The edges and the vertex are removed (their element()s can still be retrieved), and a new edge is inserted in their place. The new edge stores the specified element.

If the two incident edges of v are consistently directed, the new edge is directed in that direction. If they are both undirected, the new edge is undirected. Any other combination of directions also results in an undirected edge. Parameters

v:IVertex — the vertex to be removed
 
edgeElement:Object — the element to be stored in the new edge

Returns
IEdge — the new edge
